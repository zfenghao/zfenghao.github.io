Albion East 封闭测试云中烟霞 5 版本补丁

2023-03-08 10 min read # Albion East

云中烟霞 5 版本补丁 - 版本号 22.050.1 - 2023年3月7日
Albion East 封闭测试
Albion East 的日常维护提前一小时，现为 23:00（UTC时间）
集市上的城市之心与阵营披风冠冕将移除
基础资源重量降低的效果在被击倒后仍会保留
修正城堡与城堡哨站的宝箱在礼拜四不会重生的异常
皇家城市（Caerleon 除外）的集市上开始提供胡萝卜汤与骡子
胡萝卜汤价格：1,250 → 750
骡子价格：20,000 → 5,000
此前黄金时间为 20:00（UTC时间）的区域现改为 02:00（UTC时间）
这些区域在领地入侵日的解锁时间为 08:00（UTC时间）
此前黄金时间为 14:00（UTC时间）的区域将在领地入侵日也于 14:00（UTC时间）解锁
注意：这些时程的改变在剩余的封测公会赛季不会实装，列于此仅为维持信息透明
改动与优化
阵营战争：

固定地下城不再使阵营前线推进
在固定地下城或地下通道中击败敌对阵营的玩家时，会正确的奖励阵营点数
走私任务的增益现在将分为两部分：隐身效果（持续20秒）与沉默/禁止攻击效果（持续30秒）
这使得隐身效果结束后仍有部分效果保留的状态更加明显
在 Albion East 与 Albion West 两个服务器上，每日奖励与战役奖励都于 0:00（UTC时间）解锁
冒险者挑战：

现在在 Albion East 与 Albion West 两个服务器上，冒险者挑战皆于每个月的第一天 0:00（UTC时间）开始计算
在 Albion East 与 Albion West 两个服务器上，每日与每周奖励都于 0:00（UTC时间）解锁
征服者挑战：

现在在水晶蜘蛛项目升级所需的力量点数增为两倍
用户界面优化：

旅行商人界面中的子选单现在可滚动
优化了集市中的排序与品质过滤功能
在外观界面中增加了 "详细信息" 按钮
其他改动：

优化双人迷雾的匹配系统
对没有穿着战斗类型身体护甲（例如裸装或者穿着采集套装）的玩家施放心魔（召影法杖）时，会触发与对布甲目标施放相同的效果
时间冻结（杰出奥术法杖）现在在野怪身上最多5秒作用一次
战斗数值平衡改动
本次补丁将继续优化使用率过低的技能。铁头棍的PvE体验完整重作，其视觉效果也大幅更新，加强了沉浸感和可读性。许多表现太强的物品则略为下修。

奥术法杖

连锁飞弹在5v5水晶联赛中可以造成又稳定又高的伤害输出。为了使其他制造伤害的选项更有吸引力，其群体伤害输出能力将削弱，但对主要目标的伤害则有所提高，让它在1v1与小型战斗中的表现更好。黑暗陨石的减益效果在水晶联赛中也有些过强。为了使其破坏力更平衡，持续时间将略为缩短。

连锁飞弹（所有奥术法杖）
主要目标伤害：121 → 132
投射物弹跳伤害：60 → 25
奥术叠层伤害：61 → 55
黑暗陨石（夜祷之戒）
减益持续时间：7秒 → 5秒
斧头

裂域斧在10v10炼狱之门与类似规模的战斗中是许多秒杀组合的核心。强力余震降低生命值上限的减益状态不受防御削减上限影响，但仍然等同增加目标承受的伤害。为了降低该机制的冲击性，其减益数值有所降低。

强力余震（裂域斧）
生命值上限降低：20% → 15%
弓箭

附魔箭袋已重制为一项互动性更高的技能。更低的冷却时间使其不再着重于单点爆发，也不像以往那么害怕进攻驱散。它提供的游戏风格现在更近似于MOBA游戏中的远程射手。此外，破雾之弓的伤害略为调高，使其在大型战斗中更具表现力。

附魔箭袋（普通弓箭）
技能重制：
在接下来的8秒内，附魔6枚箭矢
当拥有附魔箭矢时，攻击速度提高50%，且普通攻击伤害提高240%
每次普通攻击消耗1枚附魔箭矢
在8秒内重复施放此技能，可以朝目标方向回避，并再次获得4枚附魔箭矢（总数无法超过6枚）
冷却时间：12秒
光明之鹰（破雾之弓）
伤害：145 → 160
匕首

冥使匕首在单人迷雾中的宰制力非常强大，本次补丁延长了暗影之刃的冷却时间，使冥使匕首利用该技能获得的机动性有所下降。在其他野怪较少的场合，若玩家使用此技能击中敌对玩家，其冷却时间会降低，与原本的冷却时间一致。

暗影之刃（所有匕首）
冷却时间：15秒 → 20秒
击中敌对玩家将使技能冷却时间降低25%
火焰法杖

由于缺乏灵活的机动力，火焰法杖在单人与小型对战中的表现一直不甚理想。为了使他们在开放地形的战斗中更具危险性，灼热烈焰的伤害输出已调升。

灼热烈焰（所有火焰法杖）
灼热烈焰投射物伤害：94 → 99
狂怒之焰投射物伤害：182 → 191
天霜法杖

为了使杰出天霜法杖在中小型对战中能成为优秀的伤害输出角色，冰雹的伤害已调高。寒冰长矛的伤害也因同样的理由调升。

寒冰长矛（所有天霜法杖）
伤害：145 → 160
冰雹（杰出天霜法杖）
伤害：130 → 142
锤子

击倒技能的沉睡效果持续时间提高。

击倒（所有锤子）
对玩家沉睡时间：4.36秒 → 5秒
对怪物沉睡时间：8.72秒 → 10秒
命中延迟：0.4秒 → 0.25秒
硬直时间：1秒 → 0.5秒
铁头棒

添加了两项新技能，以提升此武器系早期的PvE体验，并提供更多符合斗士风格的选择。阻断技能的定身时间与施放距离都已调高，使其可以有效反制开启了纯洁长袍技能的敌人。

全新Q技能：旋风打击
如旋风般挥舞长棍，持续引导0.6秒，在5米范围内每0.25秒造成38物理伤害（引导过程中可移动）
若至少击中一名敌人，则使冷却时间降低0.5秒，持续5秒（最多可堆叠3次）
冷却时间：4秒 / 引导时间：0.6秒
全新W技能：狂风之舞
朝目标方向连续挥舞武器3次，每一击造成80伤害
若成功对一名敌人命中所有3次挥击，使自身的移动速度提高30%，普通攻击伤害提高50%，持续4秒
冷却时间：12秒 / 引导时间：1秒
震荡连击（所有铁头棒）
优化前两次打击的命中判定
阻断（所有铁头棒）
施放距离：3米 → 5米
定身持续时间：1.6秒 → 2.5秒
被击退的敌人不再被减速
技能已移除：强化猛击
技能已移除：重劈

默认解锁：
震荡连击

狂风之舞
等级3：昏迷奔跑
等级15：强力挥舞
等级40：侧翻

等级85：阻断
剑

此前对双剑的增强，使其无法被打断的跳跃与随后的击飞效果非常难以反制，在单人或小型队伍的战斗中更是如此。其技能的击飞效果现在将移除。

旋转之刃（双剑）
不再击飞敌人
头盔

法师风帽的技能一直被玩家广泛使用，但其运作方式较为单调。毒药现在将被重制为一个锥形的火息术，可以同时击中多名敌人，包含隐形中的敌人，并施加持续性伤害效果。此外，不朽意志与蜘蛛丝两项技能将增强。

毒药（法师风帽）
重制为新技能：火息术
在正前方13米的锥形区域内喷吐火焰之息。被击中的敌人将会燃烧，在3秒内受到160魔法伤害
（做为参考，此前毒药的持续伤害为180）
不朽意志（迷雾行者兜帽）
冷却时间：45秒 → 30秒
蜘蛛丝（织暮者头盔）
硬直时间：0.1秒 → 0秒
投射物射程：15米 → 17米
鞋子

为了间接增强坦克在团战中的表现，挣脱的冷却时间已被延长。这使得坦克在持续时间较长的团战（像是水晶联赛）中可以通过在良好的时机打出群体控制技能得到更多价值。

挣脱（佣兵鞋子）
冷却时间：25秒 → 30秒
